O projeto

Nós vamos começar baixando e instalando as depêndencias NPM e NodeJS, logo após terminar crie uma nova pasta com o nome de scr acesse a pasta, E siga os passos abaixo, com imagem. passo 1

Com o comando npm init -y estamos criando um ambiente de desenvolvimento com node packet manager. O que siginifica que na pasta src que foi construída, terá outras subpastas que são as nossas depêndencias, como as bibiotecas que iremos baixar, é importante não apagar nenhuma dessas subpastas que o proprío NPM criou!. O único arquivo que podemos editar no futuro é o package.json Nele contém informações do seu projeto, qual é o nome da pasta do seu projeto, o nome do arquivo principal do projeto, versão no NodeJS, palavras chaves, autor do projeto, e as depêndencias instaladas no projeto, etc.
Ao executar o comando npm i express aqui você aprende como baixar uma biblioteca. Se eu pedir para você baixar uma biblioteca, você já sabe. Para darmos sequência, creio Eu! que até aqui, você criou uma pasta com o nome de src Entrou! é executou os seguintes comando! informados acima.

Vamos criar nosso primeiro servidor Hello World estamos utilizando o Framework Express versão 5.1, para construir um servidor, o Express é repleto de diversas bibliotecas. Como de criptografia, para realizar CRUD, para metódos de segurança, etc. Ops! foco, vamos criar um arquivo com o nome de index.js E, ao copiar o código acima, você criou sua primeira rota, e você pode acessar executando o comando no seu prompt node index.js para enfim, o software funcionar. Sempre utilize a nomenclatura correta, o index.js é primeiro arquivos de inicialização, dentro de um servidor. Agora abra o navegador e digite http://localhost:5500 e visualize seu servidor online.
